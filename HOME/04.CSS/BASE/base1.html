Форматы файлов

.DOC или .doc (от англ. document) - расширение имени файла, используемое для файлов,
предоставляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения
простых текстовых файлов без форматирования? однако позже стало использоваться для двоичных
форматов с разметкой.
Документ MS World

Portable Document Format (PDF) - межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с использованием ряда возможностей языка PostScript.
Документ PDF

ZIP - популяпный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет
расширение .zip и хранит в сжатом или несжатом виде один или несколько файлов, которые можно
из него извлеч путём распаковки с помощью специальной программы.
